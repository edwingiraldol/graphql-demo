# SpaceX Missions Dashboard - Product Overview

## Project Purpose
A modern web application demonstrating best practices in frontend architecture by consuming the public SpaceX GraphQL API. Built as a technical showcase for scalable React development patterns, state management, and testing strategies.

## Value Proposition
- **Educational Resource**: Demonstrates proper GraphQL consumption with Apollo Client
- **Architecture Reference**: Showcases clean separation between UI, domain, and infrastructure layers
- **Best Practices Example**: Implements Redux Toolkit, TypeScript, and modern React patterns
- **Testing Foundation**: Includes end-to-end testing with Playwright

## Key Features
- **SpaceX Data Visualization**: Browse launches, rockets, and mission details
- **Interactive Filtering**: Filter launches by year and mission status
- **Detailed Views**: Comprehensive launch and rocket detail pages
- **Responsive Design**: Tailwind CSS styling with mobile-first approach
- **Real-time Data**: Live data from SpaceX's official GraphQL API
- **State Management**: Global state handling with Redux Toolkit
- **Client-side Routing**: Multi-page navigation with React Router

## Target Users
- **Frontend Developers**: Learning modern React architecture patterns
- **Technical Interviewers**: Evaluating code quality and architectural decisions
- **Students**: Understanding GraphQL integration and state management
- **Teams**: Reference implementation for project structure and conventions

## Use Cases
- Technical portfolio demonstration
- Code review and architecture discussions
- Learning resource for Redux Toolkit and GraphQL
- Foundation for similar data-driven applications
- Interview preparation and technical assessments
